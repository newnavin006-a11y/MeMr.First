Place 64x64 (or larger) PNG piece images here named exactly:

wP.png  wN.png  wB.png  wR.png  wQ.png  wK.png
bP.png  bN.png  bB.png  bR.png  bQ.png  bK.png

White pieces must start with 'w', black with 'b', followed by piece letter (P,N,B,R,Q,K).
If no images are present the program will draw simple vector-like piece icons as a fallback.
